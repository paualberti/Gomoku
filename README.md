# Gomoku (5 in a row)

## Try it by yourself

```Bash
rm -rf ~/Gomoku/
git clone https://github.com/paualberti/Gomoku
```

## What is it?

This project is an implementation of the game Gomoku using SDL2.

The rules of the game can be found [here](https://en.wikipedia.org/wiki/Gomoku)
